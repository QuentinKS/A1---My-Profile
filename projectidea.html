Project Idea

Create a recommendation system for Twitch, a live streaming platform, that allows the user to find a specific content creator that they would like to watch. A database will group streamers thematically based on a variety of factors: country, family or mature, personality, viewer size, theme, he/she pronouns, live now and when the user inputs their wanted criteria, it would output a streamer the user may like. This in turn will allow better discovery on platforms such as Twitch. A machine learning algorithm will also be used to take feedback and then use the users experience to sort the streamer and make guesses as to what a person will like. Current discoverability on Twitch favour streamers based on viewership rather than it being tailored what the user may like.

According to Twitch Tracker, 6.9 million people stream to Twitch on a monthly basis and the platform itself has on average 2.1 million concurrent viewers. Also, with an average 87.5K concurrent streamers, it's safe to say that the market it currently saturated. It can be hard for a new streamer to be discovered in 2020 than say 2012 when there was 102K concurrent viewers, 300K monthly broadcasters and 2.2K concurrent streamers. New users would find it hard to find a streamer they can relate to and a system that recommends streamers can help to filter through all the streamers and find someone they would like to watch. With how hard it is for a new streamer on twitch to be discovered it leaves the possibility of many great content creators and streamers left in the majority because they didn’t get lucky enough, not because of their hard work or quality of content.

The system itself is an integrated feature and allow a user to find a streamer they desire in that point of time. The system is tailored to the users wants and can be reset or reuse old searches. The system utilizes machine learning to tailor searches to users as time goes on and experience is gained

The database is the root of the recommendation system. When the profile is complete the database will sort users into the categories formed by the answers to the profile questions. Streamers can be in multiple categories at once. As the user calls different categories, it will return streamers that fit into all or at least some of the categories.

The profile will ask the user for inputs using a drop-down bar which will allow the person inputting the answers to create a personalised profile. The profile itself does not need to be filled out if you are just a viewer but a streamer can fill it out or change it whenever they want. The profile will ask for information that will be attached to the user's profile and be called for when they match what an individual is looking for. The categories that will define a streamer will be language, mature or family friendly, personality, theme, he/she/they/them, pc or console. Some factors such as game/category can only be determined by what that person streams but by making use of the ‘recently streamed categories’ on Twitch channels, it's possible to make a safe assumption on what a streamer may or may not stream next.

When a user enters the site, they can choose to browse as per usual or they use the recommend feature. If they want to use the recommendation, a small window will pop-up asking for some descriptors to help the system recommend someone to the user. Youll be able to find someone based on any one of the factors or by using all of them. You can leave some blank and you can choose from all of them if you like. When recommended, you then have the option to sort by more factors like viewership (0-10, 11-50, 51-100, 101-500, 501- 1000 and 1001+ viewers), country, live or offline. This will allow you to fine tune your search. If you're looking for something to watch for a small session or if you're looking for someone whose community, you would like to become a part of.

If the user decides to leave the channel, they will be presented with a short feedback survey that will help to fine tune the recommendations. The survey will ask if they liked the recommendation and any feedback on whether the streamer, they were recommended was someone they might watch again. As feedback is gained the system will learn more and more about users experience regarding the profile they created and the streamers they have recommended. It learns which streamers are enjoyed more often and will recommend them more often, without considering, their size or partnership status. It will learn solely on the feedback of the users.

If the user is looking for someone to watch for a short amount of time or even would like to see who the system recommends, there is a feature which will randomly assign the user a content creator within the space of the parameters set to allow some variety and excitement into the mix.
